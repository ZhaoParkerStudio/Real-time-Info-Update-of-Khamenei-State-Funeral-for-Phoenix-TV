伊朗已故最高领袖哈梅内伊国葬实时简报

阿里·哈梅内伊（1939—2026）· 2月28日美以联合空袭遇害 · 国葬 2026年7月3日—9日 · 德黑兰启幕 → 库姆 → 纳杰夫/卡尔巴拉 → 马什哈德安葬

今日聚焦 / Today's Focus — 7月5日（伊朗当地）

当日 德黑兰公众悼念第二天收尾，新任最高领袖仍未现身。当地时间7月5日，伊玛目霍梅尼清真寺（大莫萨拉清真寺）公众告别仪式进入第二天。哈梅内伊另三个儿子马苏德、梅萨姆、穆斯塔法当日公开现身仪式，但新任最高领袖穆杰塔巴·哈梅内伊仍未出席任何公开场合[21]。此前其岳父（妻子之父）已代为出席7月3日遗体告别仪式[11]。伊朗政府宣布7月4—6日为公共假期配合治丧[5]。

悼念画面与现场实况。7月4日清晨6时，祈祷大厅开放，大批民众排队入内告别。灵柩前摆放一排红色郁金香，天花板悬挂纸制蝴蝶装饰；哈梅内伊及其家属灵柩上披覆伊朗国旗，灵柩旁另安放四座灵柩——分别是在空袭中一同遇难的女儿、女婿、儿媳及年仅14个月的外孙女扎赫拉（新任最高领袖穆杰塔巴的妻子与女儿）[4][6]。民众见灵柩后失声痛哭，男性循什叶派葬礼惯例有节奏捶胸哀悼，成千上万人手持象征复仇的红色旗帜，现场回荡"打倒美国""复仇"等口号[2][4]。

特朗普：本可在葬礼"一锅端"伊朗高层，但"需要有人谈判"。7月4日，特朗普接受阿克西奥斯（Axios）采访时表示，鉴于伊朗领导层集体出席葬礼，美国本可"一把"干掉对方，但"我们不会这么做，因为那样我们就找不到人谈判了"[14][13]。他对民众在葬礼上哭泣感到"惊讶"，质疑"也许全是假哭"——伊朗驻亚美尼亚使馆驳斥称"你不理解这些事，因为你既没有文明、也没有历史，没有荣誉"[13]。

官方外媒 穆杰塔巴缺席与继任机制。伊朗专家会议（Assembly of Experts）于3月8日以压倒性多数推举56岁的穆杰塔巴为第三任最高领袖，3月9日由国家媒体正式宣布；革命卫队随后发表声明表示支持[17]。反对派背景媒体伊朗国际（Iran International）披露，专家会议是在革命卫队指挥官"反复接触和心理及政治压力"下召开的，穆杰塔巴本人未亲自出席任命仪式[12]。其健康方面，据多方报道，穆杰塔巴一条腿经历三次手术可能需假肢、一只手接受手术、面部及嘴唇严重烧伤致发声困难，由总统佩泽希齐扬（心脏外科医生出身）及卫生部长直接负责医护[21]；据《Open Magazine》报道，他从不公开地点以手写信件通过摩托车/汽车人力信使链下达指令[21]。以色列国防部长此前公开将穆杰塔巴列入暗杀名单，称"任何被任命的新领袖都将是清除目标"[10]。

权力核心集体亮相，武装部队进入最高战备。自战争爆发以来长期未公开露面的革命卫队总司令瓦希迪在国葬前夕密集现身：7月2日接受官方电视台采访称"敌人会带着我们将投降的想法进坟墓，我们永不投降"，7月3日遗体告别仪式上强调"哈梅内伊是永恒的"[13]。伊朗政府宣布武装部队进入最高战备状态，悼念仪式期间德黑兰上空实施飞行管制[5]。总统佩泽希齐扬与议长卡利巴夫自战争以来首次一同露面，并在仪式期间会见外国代表团；有报道称佩泽希齐扬泪洒现场，卡利巴夫哭到身体不停颤抖[11]。卡利巴夫7月3日警告以色列企图破坏伊美谅解备忘录，并称"对美国和以色列的复仇呼声必须响彻世界"[13]。

事件背景 / Background

2026年2月28日，美国与以色列对伊朗发动联合军事打击（代号"史诗之怒"Epic Fury），以约30枚重型炸弹袭击哈梅内伊位于德黑兰的官邸"领袖之家"（Beit Rahbari），哈梅内伊及其多名亲属在空袭中遇害[5][8]。一同遇难的包括其女儿、女婿、儿媳及年仅14个月的外孙女[6]。据路透社报道，内塔尼亚胡在行动前48小时与特朗普通话，力主联合打击哈梅内伊，称这是"一生一次的机会"[20]。

遗体告别与安葬仪式原定3月在德黑兰与马什哈德举行，因战争爆发被迫中止。伊朗随后宣布全国进入40天哀悼期。直至6月下旬局势趋缓，政府重新公布完整日程，将国葬安排在7月3日至9日——这场国家级仪式在被推迟四个多月后得以重启[4]。

外国代表团与表态 / Foreign Delegations

据伊朗外交部介绍，约100个国家派遣官方代表团、公众团体或知名人士出席，包括至少8位国家元首、12个国家的议长及多国外长、部长级官员或总统特使；另有来自90多个国家的宗教界人士、学者参加[4]。伊朗外交部发言人巴加埃3日称预计近100国代表出席[5]。对公开支持美以军事行动的西欧国家，伊方未邀请[4]。

新任最高领袖穆杰塔巴动态 / New Supreme Leader Mojtaba

哈梅内伊之子、56岁的穆杰塔巴·哈梅内伊于3月8日由专家会议推举为第三任最高领袖，3月9日正式宣布，革命卫队随后表态支持[17]。但国葬期间全程未现身任何公开悼念场合，伊朗官方称"出于安全原因，穆杰塔巴不会出席葬礼"[21]。作为长子，在什叶派传统中儿子主持扶灵、祈祷是承接权力正统性的重要一步，但以色列防长早已将其列入暗杀名单，摩萨德情报网络被认为能在百万级人流中精准锁定目标[6][10]。

健康与执政方式。据多方报道，穆杰塔巴一条腿经历三次手术可能需假肢、一只手接受手术、面部及嘴唇严重烧伤致发声困难，由总统佩泽希齐扬及卫生部长直接负责医护；其首次公开声明（3月12日）以书面文本由播音员代读、无视频音频，据《耶路撒冷邮报》称文本含多处拼写错误、知情人士称"由革命卫队口授后改写"[21]。据报道，他从不公开地点以手写信件（信封密封）通过摩托车/汽车人力信使链下达指令；国际危机组织分析人士称其"无法做出关键决策或微观管理谈判"[21]。

分析认为，"高强度动员"与"低可见权力"并存意味着权力从依赖领袖个人形象转向更制度化的表达；以革命卫队为代表的安全—军事体系在国葬中作用进一步上升[4]。国葬结束后，据沙特等国渠道消息，美伊新一轮会谈将于7月11日在巴基斯坦举行，主要议题为美国对伊制裁、伊朗被冻结资金及核问题；美伊谈判因国葬暂停一周[13]。

安全态势 / Security

伊朗启动前所未有的最高级别安保预案，全国安防体系全面升级待命，政府在葬礼前夕宣布武装部队进入最高战备状态，悼念仪式期间德黑兰上空实施飞行管制[5]。德黑兰、马什哈德等城市实行准战时管理状态[4]。

空域德黑兰空域自7月3日起部分管控，7月6日全面关闭；国际和国内机场暂停运营数日[5]
灵柩防护伊朗国家电视台曾播出工人搭建灵柩展示台画面，灵柩置于防弹玻璃后展示[21]
交通市中心大部分区域交通管制，私家车禁止通行；紧急开放700多个停车区[5][4]
降温气温超35℃，清真寺附近安装6000多个高空喷淋装置为人群降温[4]
后勤军方烘焙分发逾5000万个面包，部署16个流动面包房[4]
医疗调集2500辆救护车、21架直升机、100架无人机及数千名救援人员；20多家医院、50万升静脉输液、2万间教室待命[4]
特警法新社描述德黑兰革命广场及周边部署"蒙面、重武装的特种部队"，全员高度戒备[21]

以方威胁。以色列国防部长卡茨公开警告"任何被任命的新领袖都将是清除目标"，并称以军已做好独自对伊朗采取军事行动的准备、有可能针对哈梅内伊葬礼下手[10]。据路透社报道，内塔尼亚胡在"史诗之怒"行动前48小时与特朗普通话，力主联合打击哈梅内伊，称这是"一生一次的机会"[20]。《纽约时报》报道，美方曾怀疑以色列策划暗杀伊朗谈判关键人物卡利巴夫与外长阿拉格齐，并通过第三国向伊朗预警；4月卡利巴夫返程时其专机因情报显示以军战机进入伊朗领空而紧急降落马什哈德[5]。伊朗武装部队哈塔姆安比亚中央总部司令阿卜杜拉希7月2日警告："对我国的任何威胁与侵略都将招致武装部队严厉、令其清醒的回击"[13]。截至7月5日，未检索到国葬仪式期间发生实际袭击或重大安全事件的官方确认报道。

人群规模 / Crowd Size

官方口径 德黑兰副市长（社会文化事务）塔瓦科利扎德此前预估首都一地将有1500万至2000万人参与；德黑兰市长扎卡尼预测7月6日送葬游行约2000万人，称为"该市历史上规模最大的集会"[7][5]。官方预计未来几天总参与人数可能达数千万，有意与1989年霍梅尼葬礼（约1000万人）的历史规模形成对照[4]。

外媒/通讯社口径 新华社、央视现场报道称7月4—5日"数百万"民众参与悼念[2][3]；法新社等通讯社多援引伊方公布的预估数字并标注为"伊朗官方预计"，对最终实到人数持保留，未给出独立点算的现场人数。路透社画面显示街道人山人海。需注意：官方数千万的预估为"未来几天累计"且含政治象征成分，与单日实到人数口径不同，同传解说时应区分表述。

国际反应与安理会动态 / International Reaction

美国。特朗普2月28日发文称哈梅内伊是"历史上最邪恶的人之一"；7月4日接受阿克西奥斯采访称本可在葬礼"一锅端"伊朗高层但选择不这样做，因为"需要有人进行谈判"，并对民众悲恸感到意外、质疑哭泣"也许是假的"[14][13]。美国国务卿鲁比奥6月26日下令驻外使领馆动用外交资源施压所在国，将出席葬礼视为"不友好"举动；据媒体统计，至少13个国家因此取消出席[6]。

美伊谈判。国葬正值美伊60天停火期，谈判因国葬暂停一周。7月1日双方在多哈完成低级别间接技术性会谈，议题涉及解冻伊朗海外资金及霍尔木兹海峡通行费[5]。据沙特等国渠道消息，美伊新一轮会谈将于7月11日在巴基斯坦举行[13]。

联合国安理会。3月1日安理会应法国总统马克龙呼吁召开紧急会议，秘书长古特雷斯"对称谴责"美以打击与伊朗报复性打击；会议"无任何成果、无决议付诸表决"（美国作为常任理事国拥有否决权）[18]。英、法、德已启动第2231号决议"快速恢复"（snapback）机制，恢复此前中止的六项涉伊安理会决议（含武器禁运）[18]。海合会成员国向安理会提交决议草案谴责伊朗袭击[18]。

中国。事发当日外交部表态"高度关切"，强调"伊朗的主权、安全和领土完整应得到尊重"；3月5日外交部副部长苗得雨专程赴伊朗驻华使馆吊唁[15]；国葬期间何维代表中方出席并会见伊方领导人[15]。

俄罗斯与欧盟。普京致唁电称哈梅内伊遇害是"对人类道德准则和国际法的公然践踏"[16]；梅德韦杰夫作为特使出席葬礼。欧盟委员会主席冯德莱恩称"哈梅内伊离去，伊朗人民有了新的希望……这一时刻也带来不稳定风险"；欧盟外交高级代表卡拉斯称"现在有一条通往不同伊朗的开放道路"[19]。英、法、德联合声明对伊朗"鲁莽的"报复性打击感到"震惊"。
