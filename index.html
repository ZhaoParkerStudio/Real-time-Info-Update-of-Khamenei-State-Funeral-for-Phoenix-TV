伊朗已故最高领袖哈梅内伊国葬实时简报

阿里·哈梅内伊（1939—2026）· 2月28日美以联合空袭遇害 · 国葬 2026年7月3日—9日 · 德黑兰启幕 → 库姆 → 纳杰夫/卡尔巴拉 → 马什哈德安葬

今日聚焦 / Today's Focus — 7月5日

当日 德黑兰公众悼念进入第二天。当地时间7月5日8时许，数千民众聚集在德黑兰伊玛目霍梅尼清真寺（大莫萨拉清真寺），为哈梅内伊及其在空袭中遇害的四名亲属举行葬礼礼拜。伊朗政府、军方官员以及部分外国代表团出席该仪式，伊斯兰革命卫队总司令瓦希迪也出现在礼拜现场[3]。伊朗政府已宣布7月5日（及前后）为公共假期，配合治丧[7]。

悼念画面与现场实况。7月4日清晨6时，伊玛目霍梅尼清真寺祈祷大厅开放，大批民众排队入内告别。灵柩前摆放一排红色郁金香，天花板悬挂纸制蝴蝶装饰。身着黑衣的悼念者不断将围巾等随身物品抛向灵柩（伊朗宗教习俗，借此为逝者祈福）。哈梅内伊及其家属灵柩上披覆伊朗国旗，灵柩旁另安放四座灵柩——分别是在袭击中一同遇难的女儿、女婿、儿媳及年仅14个月的外孙女扎赫拉（即新任最高领袖穆杰塔巴的妻子与女儿）[4][6]。许多民众见灵柩后失声痛哭，男性循什叶派葬礼惯例有节奏捶胸哀悼，成千上万人手持象征复仇的红色旗帜，现场回荡"打倒美国""复仇"等口号[4]。

官方外媒 新任最高领袖穆杰塔巴全程缺席。伊朗官方解释称，穆杰塔巴本人执意出席父亲葬礼，但安全部门以局势紧张、无法保障其人身安全为由强行劝阻其公开露面，并称其已私下瞻仰过父亲遗体[6]。此前以色列国防部长公开将穆杰塔巴列入暗杀名单，称"任何被任命的新领袖都将是清除目标"[10]。这是该国四十五年宪政史上首次最高领袖缺席国家级核心丧葬仪式。

权力核心集体亮相。自战争爆发以来长期未公开露面的伊斯兰革命卫队总司令瓦希迪，在国葬前夕出席多场活动：7月2日参加筹备会议并在哈梅内伊故居附近小规模悼念仪式上坐于灵柩旁守灵；7月3日遗体告别仪式上公开表示"哈梅内伊是永恒的"，誓言绝不向敌对势力屈服；7月5日再度现身葬礼礼拜现场[3][4]。总统佩泽希齐扬与议长卡利巴夫也自战争以来首次一同露面，并在仪式期间会见外国代表团[4]。有报道称佩泽希齐扬在告别仪式上泪洒现场，议长卡利巴夫哭到身体不停颤抖[11]。

在伊朗东北部圣城马什哈德（哈梅内伊出生地）举行遗体送葬及安葬仪式，安葬于伊玛目礼萨圣陵[1][7]。仪式安排在伊斯兰历穆哈兰姆月、接近第四位什叶派伊玛目萨贾德殉难纪念日前后完成[4]。

事件背景 / Background

2026年2月28日，美国与以色列对伊朗发动联合军事打击（代号"传奇之怒"），以约30枚重型炸弹袭击哈梅内伊位于德黑兰的官邸"领袖之家"（Beit Rahbari），哈梅内伊及其多名亲属在空袭中遇害[5][8]。一同遇难的包括其女儿、女婿、儿媳及年仅14个月的外孙女[6]。

遗体告别与安葬仪式原定3月在德黑兰与马什哈德举行，因战争爆发被迫中止。伊朗随后宣布全国进入40天哀悼期。直至6月下旬局势趋缓，政府重新公布完整日程，将国葬安排在7月3日至9日——这场国家级仪式在被推迟四个多月后得以重启[4]。

外国代表团与表态 / Foreign Delegations

据伊朗外交部介绍，约100个国家派遣官方代表团、公众团体或知名人士出席，包括至少8位国家元首、12个国家的议长及多国外长、部长级官员或总统特使[4]。伊朗外交部发言人巴加埃3日称预计近100国代表出席[5]。

新任最高领袖穆杰塔巴动态 / New Supreme Leader Mojtaba

哈梅内伊之子、56岁的穆杰塔巴·哈梅内伊依其父生前签署的权力移交备忘录继任最高领袖，但全程未现身任何公开悼念场合[6]。作为长子，在什叶派传统中儿子主持扶灵、祈祷是承接权力正统性的重要一步，数百万民众原本期待亲眼见证，但以色列防长早已将其列入暗杀名单，摩萨德情报网络被认为能在百万级人流中精准锁定目标[6]。

伊朗官方统一说辞为：穆杰塔巴执意出席，但安全部门以局势紧张、无法保障其人身安全为由强行劝阻，并称其已私下瞻仰过父亲遗体完成悼念[6]。其岳父（妻子之父）现身仪式现场[11]。

分析认为，"高强度动员"与"低可见权力"并存意味着权力从依赖领袖个人形象转向更制度化的表达；以革命卫队为代表的安全—军事体系在国葬中作用进一步上升[4]。

安全态势 / Security

伊朗启动前所未有的最高级别安保预案，全国安防体系全面升级待命。德黑兰、马什哈德等城市实行准战时管理状态[4]。

空域德黑兰空域自7月3日起部分管控，7月6日全面关闭；国际和国内机场暂停运营数日[5]

安置400多个帐篷搭建临时安置营地，清真寺、学校改造为临时接待点[6][5]

以方威胁。以色列国防部长公开警告"任何被任命的新领袖都将是清除目标"[10]。《纽约时报》报道，美方曾怀疑以色列策划暗杀伊朗谈判关键人物卡利巴夫与外长阿拉格齐，并通过第三国向伊朗预警；4月卡利巴夫返程时其专机因情报显示以军战机进入伊朗领空而紧急降落马什哈德[5]。

人群规模 / Crowd Size

官方口径 德黑兰副市长（社会文化事务）塔瓦科利扎德此前预估首都一地将有1500万至2000万人参与；德黑兰市长扎卡尼预测7月6日送葬游行约2000万人，称为"该市历史上规模最大的集会"[7][5]。官方预计未来几天总参与人数可能达数千万，有意与1989年霍梅尼葬礼（约1000万人）的历史规模形成对照[4]。

外媒/通讯社口径 新华社、央视现场报道称7月4—5日"数百万"民众参与悼念[2][3]；法新社等通讯社多援引伊方公布的预估数字并标注为"伊朗官方预计"，对最终实到人数持保留。路透社画面显示街道人山人海。需注意：官方数千万的预估为"未来几天累计"且含政治象征成分，与单日实到人数口径不同，同传解说时应区分表述。

国际反应与谈判动态 / International Reaction

美国。特朗普称美国本可在葬礼期间袭击出席的伊朗官员，但选择不这样做，因为"需要有人进行谈判"；他对民众的悲恸情绪感到意外，称原以为伊朗人憎恨哈梅内伊，并质疑现场哭泣"也许是假的"[4]。美国国务卿鲁比奥6月26日下令驻外使领馆动用外交资源施压所在国，将出席葬礼视为"不友好"举动；据媒体统计，至少13个国家因此取消出席（含5个非洲、3个东欧国家）[6]。

美伊谈判。国葬正值美伊60天停火期，谈判因国葬暂停一周。7月1日双方在多哈完成低级别间接技术性会谈，议题涉及解冻伊朗海外资金及霍尔木兹海峡通行费[5]。有消息称双方同意解冻存放卡塔尔的30亿美元伊朗资金用于人道采购，但美方否认达成共识[5]。

联合国与国际。巴西谴责局势升级并呼吁尊重《联合国宪章》；中国与俄罗斯谴责侵犯伊朗主权[12]。伊朗议长卡利巴夫公开表示"对美国和以色列的复仇呼声必须响彻世界"，并警告以色列企图破坏伊美谅解备忘录[4][5]。
